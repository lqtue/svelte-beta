# VWAI / CDEC — Legacy / Deprioritized

This directory contains the VWAI (Vietnam War Archive Initiative) and CDEC integration components. The VWAI collaboration has ended and this work is deprioritized.

**Status:** Not actively developed. Code retained for reference only.

**Do not delete** the `src/routes/vwai/` route or this library — it may be reactivated later. But do not add new features or link to it from the public navigation.

The admin API routes at `src/routes/api/admin/cdec/` reference `vwai_member` role checks and should stay as-is.
